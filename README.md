# SeedPalette

Where kids’ creativity begins and colors grow.

## Run

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Features

- 50 built-in coloring pages (5 categories x 10)
- Brush, eraser, bucket fill (flood fill)
- Undo/redo (20 snapshots)
- Photo to coloring page (offline edge detection pipeline)
- IndexedDB gallery persistence
- Export artwork to A4 PDF with jsPDF
- Surprise Me random page picker
